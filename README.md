# ☕ Artefakt Project ☕

>**Last update:** January 2026  

An Android social media platform built for artists and art lovers to interact and share art.

## ✨ Features
- ⭐ Upload images and text — create posts
- ⭐ Like, share, and comment on posts
- ⭐ Follow and unfollow users
- ⭐ Search for accounts and posts
- ⭐ **Culturalize Me** — search for information about anything related to art
- ⭐ Personalized feed and home page
- ⭐ Direct messaging between users

## 💻 Technologies Used
- Flutter
- SQL (Database) + Supabase
- Android Studio (Emulator)
